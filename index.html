1. Introdução

Android Studio é a IDE oficial para o desenvolvimento de apps Android. Baseado em IntelliJ, ele oferece ferramentas poderosas para criar, testar e publicar aplicativos.

2. Instalação

Baixe pelo site oficial: developer.android.com/studio

Execute o instalador e siga as instruções padrão.

3. Criando seu primeiro projeto

  1. Abrir o Android Studio
  2. Clicar em "New Project"
  3. Escolher "Empty Activity"
  4. Nomear e escolher a linguagem (Kotlin)
  5. Clicar em "Finish"

4. Estrutura do Projeto

  • MainActivity.kt: código principal
  • activity_main.xml: layout visual
  • AndroidManifest.xml: configurações do app

6. Rodando no Emulador

Abra "Device Manager", crie um novo dispositivo virtual, escolha uma imagem de sistema e clique em "Play".
